# 언어 정책

## 중요 언어 { #important-languages }

| 우선순위 | 언어 | 역할 |
|---|---|---|
| P0 | English | 콘텐츠 원본이며 모든 사양 변경을 먼저 반영 |
| P1 | 日本語 | 유지관리자의 주 언어로 영어와 같은 릴리스에서 전체 갱신 |
| P1 | 简体中文 | CJK PDF, 대체 글꼴, OCR 사용 사례와 직접 연결 |
| P1 | 한국어 | CJK PDF와 Python 사용자에게 중요 |

번역이 완성되고 strict 빌드를 통과한 언어만 언어 선택기에 공개합니다.

## 유지보수 규칙 { #translation-principles }

- API 이름, 클래스명, 인자명, 코드는 영어를 유지합니다.
- 직역보다 기술적 의미와 자연스러운 표현을 우선합니다.
- 모든 언어가 같은 제목 ID를 사용해 링크를 안정적으로 유지합니다.
- 버전, 성능 수치, 보안 지침은 같은 변경에서 동기화합니다.
- 새 콘텐츠는 영어 원본을 먼저 수정한 뒤 P1 언어를 갱신합니다.

전체 선정 근거, 승격 조건, 출처는 저장소의
[`LANGUAGES.md`](https://github.com/yhay81/pylopdf/blob/main/LANGUAGES.md)를 참고하세요.
